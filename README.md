# 📅 Planner Digital Profissional 2026

Este é um **Planner Digital Inteligente** desenvolvido para funcionar como um Aplicativo Progressivo (**PWA**) em tablets e smartphones. Ele combina organização diária, sincronização na nuvem e inteligência artificial para otimização de notas e produtividade.

---

## ✨ Funcionalidades

* **☁️ Sincronização Cloud:** Salva automaticamente suas notas e prioridades através do **Firebase**.
* **🤖 Inteligência Artificial (Gemini):**
    * Botão para organizar notas de reunião profissionalmente.
    * Extração automática das 5 prioridades mais importantes do seu texto.
* **📱 Interface Otimizada para Tablet:**
    * Visual de papel pautado para escrita confortável.
    * Navegação rápida por meses na barra lateral.
    * Suporte a gestos (*Swipe*) para trocar de dia deslizando o dedo.
* **🚀 PWA Ready:** Pode ser instalado como um aplicativo nativo no Android e iOS, funcionando em tela cheia.

---

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML5, CSS3 e **Tailwind CSS** (design moderno e responsivo).
* **Lógica:** JavaScript ES6+ e integração de APIs.
* **Backend:** **Firebase Firestore** para persistência de dados em tempo real.
* **IA:** **Google Gemini API** para processamento inteligente de texto.

## 🚀 Como instalar no seu Tablet

Como este repositório está no GitHub, você pode usar o **GitHub Pages** para hospedar a sua agenda:

1.  No seu repositório no GitHub, vá em **Settings** (Configurações).
2.  No menu lateral, clique em **Pages**.
3.  Em "Build and deployment" > "Branch", selecione `main` e a pasta `/ (root)`. Clique em **Save**.
4.  Aguarde cerca de 1 a 2 minutos. O GitHub fornecerá um link (ex: `https://seu-usuario.github.io/agenda/`).
5.  Abra esse link no navegador do seu Tablet:
    * **Android (Chrome):** Clique nos três pontos e selecione **"Instalar aplicação"**.
    * **iPad (Safari):** Clique no botão de compartilhar e selecione **"Adicionar ao Ecrã Principal"**.
